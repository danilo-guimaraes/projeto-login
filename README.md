---

## ✨ Funcionalidades

- Formulário de login com campos de **e-mail** e **senha**
- Validação nativa HTML5 (`required`, `minlength`, `maxlength`)
- Link para recuperação de senha
- **Layout responsivo** com Media Queries para 3 breakpoints:
  - 📱 Mobile (< 768px)
  - 📟 Tablet (768px – 992px)
  - 🖥️ Desktop (> 992px)
- Ícones via **Google Material Symbols**
- Gradiente de fundo dinâmico por breakpoint

## 🧠 Conceitos Praticados

| Conceito | Descrição |
|---|---|
| Semântica HTML5 | `<main>`, `<section>`, `<form>`, `<label>` |
| Formulários | `autocomplete`, `required`, `minlength`, `maxlength` |
| CSS Float | Layout com `float: left/right` |
| Media Queries | `@media screen and (min-width: ...)` |
| Google Fonts API | Material Symbols Outlined |
| CSS Gradients | `linear-gradient` com direções diferentes por breakpoint |
